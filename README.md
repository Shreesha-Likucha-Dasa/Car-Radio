# CarRadio - Android Auto Internet Radio

A modern, high-performance internet radio application built with **Media3** and designed specifically for **Android Auto**. 

## 🚀 Features
* **Global Radio Access:** Powered by the RadioBrowser API to fetch thousands of stations worldwide.
* **Android Auto Native UI:** Full integration with the car's dashboard for safe, driver-friendly browsing.
* **Persistent Favorites:** Save your favorite stations using a local JSON-based storage system.
* **Context-Aware Navigation:** Skip through stations using the car's steering wheel or screen controls based on your current browsing context (Local, Search, or Favorites).
* **Smart Search:** Real-time search by station name, country, or genre.
* **Audio Focus Management:** Handles phone calls and navigation interruptions gracefully.

## 🛠 Tech Stack
* **Language:** Kotlin
* **Media Framework:** Jetpack Media3 (ExoPlayer & MediaSession)
* **Networking:** Retrofit / OkHttp
* **Data Parsing:** Gson
* **Architecture:** Coroutines for asynchronous API fetching

3. Build the project in Android Studio.
4. Use the **Desktop Head Unit (DHU)** to test Android Auto features.
